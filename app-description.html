🎯 Application Overview

This is an innovative AI-powered Arabic language learning application specifically designed for Quran memorizers (huffaz) who speak English. The application helps users practice daily Arabic conversation using Quranic phrases and verses they have already memorized. It combines their existing Quranic knowledge with practical language skills through interactive exercises, phrase management, and progress tracking.

Unique Approach: Unlike traditional Arabic learning apps, this platform leverages the user's existing Quranic knowledge as a foundation for building conversational Arabic skills, making the learning process more efficient and meaningful.

🎓 Exercise Types

The application offers seven distinct types of interactive exercises designed to reinforce different aspects of Arabic language learning:

🔄Substitution Drill

Replace words in Quranic phrases to understand structure and meaning. Helps grasp grammar patterns and word relationships.

💬Daily Conversation

AI asks questions in English, you respond using appropriate Quranic expressions. Practice practical communication skills.

🧩Completion Drill

Complete verses when given the beginning, testing your memorization and understanding of Quranic context.

⚖️Comparison

Explain differences between similar verses to deepen comprehension and analytical skills.

🎭Role Play

Use Quranic verses in real-life emotional and interactive scenarios. Apply spiritual knowledge to daily situations.

🔄Transformation

Convert statements into questions and vice versa using Quranic Arabic. Master grammatical transformations.

🎯Thematic Questions

Find appropriate Quranic verses for specific life situations. Develop wisdom and contextual understanding.

🔧 Technical Architecture

Frontend Technology Stack

Framework: React with TypeScript using Vite for development and building

Routing: Wouter for client-side routing

UI Components: Shadcn/ui component library built on Radix UI primitives

Styling: Tailwind CSS with CSS variables for theming and Arabic font support (Amiri)

State Management: TanStack Query (React Query) for server state management

Speech Synthesis: Browser Web Speech API for Arabic text-to-speech functionality

Backend Technology Stack

Server: Express.js with TypeScript running in ESM mode

Database: PostgreSQL with Drizzle ORM for type-safe operations

API Design: RESTful API with structured routes

Authentication: JWT tokens with bcrypt password hashing

Data Validation: Zod schemas for request/response validation

Payment Gateway: HyperPay integration for secure payment processing

Database Schema

  • Users Table: Stores user accounts with secure password hashes
  • Phrases Table: Stores Arabic text, English translations, Surah references, and categorization
  • User Progress: Tracks mastery levels and practice statistics per phrase
  • Exercise Sessions: Records individual exercise attempts and results
  • Daily Statistics: Aggregates daily usage and performance metrics
  • Question Banks: Stores exercise questions mapped to phrases

🔐 Security Features

  • Password Security: Bcrypt hashing with 12 salt rounds for secure password storage
  • Authentication: JWT (JSON Web Tokens) for stateless session management
  • Environment Variables: Secure credential storage for sensitive API keys and database connections
  • Payment Security: HyperPay integration with PCI-compliant payment processing
  • Data Validation: Zod schema validation on all API endpoints to prevent injection attacks

📈 Learning Process Flow

Step 1: Profile Creation

Users sign up and provide information about their Quran memorization level and Arabic learning goals. The system assesses current knowledge and customizes the learning path accordingly.

Step 2: Access Quranic Database

Users explore the comprehensive database of Quranic sentences organized by complexity and usage. Each verse comes with Arabic text, English translation, and real-life context examples.

Step 3: Practice with AI

Users engage in interactive exercises and conversations with the AI tutor. Practice pronunciation, sentence construction, and use Quranic expressions in daily scenarios.

Step 4: Track Progress

Users monitor their learning journey with detailed analytics and weekly reports. See which verses have been mastered and receive recommendations for continued learning.
