يجب تعديل الكود ليعرض ترجمة الكلمة المستهدفة بناءً على لغة المتصفح أو التطبيق المختارة، بالإضافة إلى تعريب/ترجمة ترويسة السورة.

إليك الحل التقني واللغوي المباشر بصيغة "Replit" (منطق برمجي صارم) لتنفيذ هذا التغيير:

1. هيكلة البيانات (Data Structure)
لا يمكنك الاعتماد على الصور فقط. يجب أن يحتوي ملف البيانات (JSON أو قاعدة البيانات) على معنى الكلمة بكل اللغات المدعومة.

التعديل المطلوب في ملف exercises.js أو قاعدة البيانات:

JavaScript

2. واجهة المستخدم (The Component Logic)
في واجهة العرض (Frontend)، يجب استبدال الصورة الثابتة أو النص العربي فقط بنص يجمع بين (الكلمة العربية + معناها بلغة المستخدم).

منطق الكود المقترح (React/JSX):

JavaScript
// افتراض أن 'lang' هو متغير يحمل لغة الصفحة الحالية (en, id, tr, ar)
const currentExercise = exercises[currentIndex];
const translatedWord = currentExercise.targetWordTranslation[lang];

// 1. ترجمة الترويسة (Header)
const headerText = lang === 'ar' 
  ? `سورة ${currentExercise.surahName} — الآية ${currentExercise.verseNumber}`
  : lang === 'id'
  ? `Surah ${currentExercise.surahName} — Ayat ${currentExercise.verseNumber}`
  : `Surah ${currentExercise.surahName} — Verse ${currentExercise.verseNumber}`; // Default English

// 2. عرض السؤال مع الترجمة
return (
  <div className="exercise-container">

        <span className="translated-meaning">({translatedWord})</span>

3. الترجمات المعتمدة (للتطبيق الفوري)
لتقوم بتحديث المحتوى فوراً، إليك الترجمات الدقيقة للكلمات التي طلبتها بناءً على سياق الآيات:

أ) كلمة: "أَنزَلْنَاهُ" (سورة القدر)

English: We sent it down / We revealed it

Indonesian: Kami menurunkannya

Context: تشير إلى إنزال القرآن.

ب) كلمة: "الْفَلَقِ" (سورة الفلق)

English: The Daybreak / The Dawn

Indonesian: Waktu Subuh

Context: الصبح الذي ينفلق منه الضوء.

ج) كلمة: "تَبَّتْ" (سورة المسد)

English: Perish / Ruined

Indonesian: Binasalah / Celakalah

Context: دعاء بالهلاك والخسران.

د) ترويسة الآية (Header Format):

English: 📖 Surah Al-Qadr — Verse 1

Indonesian: 📖 Surah Al-Qadr — Ayat 1

Arabic: 📖 سورة القدر — الآية 1

الخلاصة "بصرامة":
لن يتعلم المستخدم اللغة إذا كان يطابق أشكالاً (صوراً) فقط. يجب ربط المتغير targetWord بمصفوفة ترجمات translations واستدعاء الترجمة بناءً على locale الصفحة. هذا هو الفارق بين "لعبة مطابقة صور" وبين "تطبيق تعليم لغة".
